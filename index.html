Qu'est-ce que CahierSmart ?

CahierSmart est une application web de gestion des cahiers de textes scolaires. Elle remplace le support papier par une interface numérique centralisée, accessible depuis n'importe quel appareil.

Objectif principal

Centraliser la saisie et la consultation des contenus pédagogiques en temps réel.

Interconnexion

Enseignants, élèves, parents et admins partagent un espace commun sécurisé.

Multi-device

Interface responsive disponible sur ordinateur, tablette et mobile.

1
Cloner le dépôt
git clone https://github.com/votre-org/cahiersmart.git
cd cahiersmart
2
Installer les dépendances
npm install
3
Configurer l'environnement
cp .env.example .env
# Renseigner DATABASE_URL, JWT_SECRET...
4
Lancer les migrations
npx prisma migrate dev
5
Démarrer l'application
npm run dev

Gestion des rôles

CahierSmart utilise un système RBAC (Role-Based Access Control) avec 4 profils distincts.

RôleSaisie coursVoir notesGérer usersStats globales
EnseignantOuiSa classeNonNon
EleveNonSes notesNonNon
ParentNonSon enfantNonNon
AdminOuiOuiOuiOui

Endpoints principaux

GET/api/cahiersLister tous les cahiers de textes
POST/api/cahiersCréer une nouvelle entrée
GET/api/cahiers/:idDétail d'une séance
PUT/api/cahiers/:idModifier une séance
DELETE/api/cahiers/:idSupprimer une séance
POST/api/auth/loginAuthentification → JWT token
GET/api/users/meProfil de l'utilisateur connecté
GET/api/stats/classesStatistiques par classe (admin)
